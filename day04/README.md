# Day 4 - Dictionaries

Сегодня я изучил словари (dict) в Python.

## Что изучено

* Создание словарей
* Ключи и значения
* Получение данных по ключу
* Изменение значений
* Добавление новых ключей
* Метод `.get()`
* Ошибка `KeyError`

## Примеры

Создание словаря:

```python
user = {
    "name": "Artem",
    "age": 14,
    "language": "Python"
}
```

Получение данных:

```python
print(user["name"])
```

Изменение данных:

```python
user["age"] = 15
```

Добавление новых данных:

```python
user["city"] = "Dnipro"
```

## Что понял

Словари удобно использовать, когда один объект имеет несколько свойств.

Например:

* Пользователь: имя, возраст, город.
* Задача: название, статус выполнения.
* Товар: название, цена, количество.
