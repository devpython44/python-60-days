# День 8 — RPG: сохранения и магазин

Сегодня в RPG были добавлены новые возможности:

## Что реализовано

### Сохранение и загрузка героя

* Сохранение данных в `player.json`
* Автоматическая загрузка при запуске игры
* Обработка отсутствия файла сохранения

### Магазин

* Покупка зелья лечения (+20 HP)
* Улучшение урона (+5)
* Проверка количества золота перед покупкой
* Автоматическое сохранение после покупки

### Уже есть в игре

* Герой с характеристиками
* Уровни и опыт
* Золото
* Исследование леса
* Случайные события
* Бой с волком
* Система наград
* Сохранение прогресса

## Планы

* Новые монстры
