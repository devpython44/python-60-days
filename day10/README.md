# День 10 — Система уровней и первый босс

Сегодня была улучшена система развития персонажа и добавлен первый босс.

## Что изменено

### Система повышения уровня

Создана отдельная функция:

```python
check_level_up()
```

Теперь проверка уровня выполняется автоматически после получения опыта.

Повышение уровня работает:

* после исследования леса;
* после победы над монстрами;
* будет работать для будущих боссов.

При повышении уровня герой получает:

* +20 HP
* +5 урона

### Первый босс

В игру добавлен первый босс:

**Вожак волков**

Характеристики:

* HP: 100
* Урон: 30
* Награда: 100 золота
* Награда: 100 опыта

Для босса используется та же универсальная система боя, что и для обычных противников.

### Улучшение структуры кода

* Логика повышения уровня вынесена в отдельную функцию.
* Код стал проще поддерживать.
* Подготовлена основа для добавления новых боссов и врагов.
