# День 11 — Инвентарь и трофеи

Сегодня в RPG появилась первая система предметов.

## Что добавлено

### Инвентарь

У героя появился собственный инвентарь.

Теперь предметы сохраняются вместе с персонажем в файле сохранения.

Добавлена функция просмотра инвентаря.

### Трофеи с монстров

После победы над волком может выпасть предмет:

* Клык волка

Предмет автоматически добавляется в инвентарь.

### Продажа предметов

В магазин добавлена возможность продавать трофеи.

Сейчас можно продать:

* Клык волка — 10 золота

После продажи предмет удаляется из инвентаря, а игрок получает золото.

### Улучшение боевой системы

Функция боя была переработана.

Теперь используется универсальная функция:

```python
fight_enemy(enemy)
```

Она позволяет использовать одну систему боя для:

* обычных монстров;
* боссов;
* будущих противников.

Добавлена функция случайного боя:

```python
random_fight()
```

И отдельный бой с боссом:

```python
fight_boss()
```
