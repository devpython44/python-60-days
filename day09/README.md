# День 9 — Новые монстры и улучшение боевой системы

Сегодня RPG получила первое серьёзное обновление боевой системы.

## Что добавлено

### Новые противники

В игру добавлены новые монстры:

* Гоблин
* Волк
* Орк

Каждый противник имеет собственные характеристики:

* HP
* Урон
* Награду золотом
* Награду опытом

### Универсальная система боя

Ранее существовала отдельная функция боя только с волком.

Теперь используется одна функция:

```python
fight_enemy()
```

Она работает с любым монстром и автоматически использует его характеристики.

### Случайный противник

Перед началом боя игра случайным образом выбирает одного из доступных врагов.

Это делает каждый бой уникальным.

### Улучшение структуры кода

* Убрано дублирование кода
* Монстры вынесены в отдельные словари
* Используется копия монстра во время боя
* Награды берутся из данных самого противника
