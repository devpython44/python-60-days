# Day 3 - Functions

Сегодня я изучил функции в Python и применил их в своем проекте TODO Manager.

## Что изучено

* Создание функций через `def`
* Вызов функций
* Параметры и аргументы
* Возврат значений через `return`
* Область видимости переменных
* Рефакторинг кода

## Что добавлено в проект

* Функция `add_task()`
* Функция `show_tasks()`
* Функция `delete_task()`
* Функция `save_tasks()`

## Улучшения

Код стал более читаемым и структурированным. Вместо большого количества кода внутри меню логика была вынесена в отдельные функции.

### Было

```python
if client_data == "1":
    ...
elif client_data == "2":
    ...
elif client_data == "3":
    ...
```

### Стало

```python
if client_data == "1":
    add_task()

elif client_data == "2":
    show_tasks()

elif client_data == "3":
    delete_task()
```

## Чему я научился

* Разбивать программу на небольшие части
* Использовать функции для повторяющихся действий
* Передавать данные через параметры
* Возвращать результат через `return`
* Делать код более профессиональным и удобным для поддержки
