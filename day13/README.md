# День 13 — Начало проекта Python Mentor

Сегодня начал разработку собственного обучающего приложения **Python Mentor** на Python с использованием ООП и Tkinter.

## Что сделано

### Изучение ООП

Создан класс:

* `Lesson`

Класс хранит:

* название урока;
* теоретический материал;
* список задач.

Пример создания урока:

```python
lesson_functions = Lesson(
    "Функции",
    "Теория по функциям"
)
```

### Создание GUI

Создано первое графическое окно на Tkinter.

Добавлены:

* заголовок приложения;
* кнопка "Функции";
* кнопка "Списки";
* область отображения теории.

### Работа с уроками

Реализован показ теории по нажатию на кнопку.

Доступные уроки:

* Функции
* Списки

Теория отображается непосредственно в окне приложения.

## Чему научился

* создавать классы;
* использовать `__init__`;
* работать с объектами;
* хранить данные внутри объектов;
* создавать окна в Tkinter;
* создавать кнопки и надписи;
* связывать ООП и графический интерфейс.
