Мой путь изучения Python за 60 дней.

Сегодня я начал новый проект — текстовую RPG игру.

### Что реализовано

* Герой с характеристиками:

  * Имя
  * HP
  * Золото
  * Урон
  * Уровень
  * Опыт

* Главное меню

* Исследование леса

* Случайные события

* Получение золота

* Получение опыта

* Система уровней

* Повышение характеристик героя

* Бой с волком

* Награды за победу

* Смерть героя

### Чему научился

* Использовать модуль `random`
* Создавать игровые объекты через словари
* Работать с циклами `while`
* Использовать оператор `and`
* Делать простую боевую систему
* Создавать систему опыта и уровней

Проект продолжает развиваться.
