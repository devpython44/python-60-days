# День 12 — Первое знакомство с ООП

Сегодня начал изучать объектно-ориентированное программирование (ООП).

## Что изучено

### Классы

Создал первый класс:

```python
class Player:
```

### Конструктор

Изучил метод:

```python
__init__()
```

который автоматически вызывается при создании объекта.

### Атрибуты объекта

Научился создавать характеристики через:

```python
self.name
self.hp
self.damage
```

### Создание объектов

Создал несколько объектов одного класса:

```python
player1 = Player("Artem", 100, 10)
player2 = Player("Ivan", 120, 15)
```

### Методы класса

Создал первый метод:

```python
def show(self):
```

который выводит характеристики объекта.

Также познакомился с методом получения урона:

```python
def take_damage(self, damage):
```

## Что понял

ООП позволяет один раз описать структуру объекта и затем создавать любое количество экземпляров без копирования кода.
